# Bulls and Cows

Eine Java-Konsolenanwendung, in der der Spieler einen geheimen Code aus eindeutigen Symbolen erraten muss.  
Nach jedem Versuch gibt das Programm Feedback in Form von **Bulls** (richtige Position) und **Cows** (falsche Position, richtiges Symbol), um den Spieler schrittweise näher an die Lösung zu bringen.

## Tech Stack

- Java  
- IntelliJ IDEA Ultimate

## Features

- Validierung der Eingaben (Code-Länge und Anzahl möglicher Symbole)  
- Unterstützung von 36 Symbolen (0–9, a–z)  
- Erzeugung eines geheimen Codes mit einzigartigen Zeichen  
- Darstellung des Codes als `**** (0–9, a-f)`  
- Rundenbasiertes Gameplay mit Turn-Anzeige  
- Automatische Auswertung von Bulls und Cows  
- Beenden des Programms bei ungültiger Eingabe  
- Vollständig spielbares Bulls-and-Cows-Konsolenspiel
